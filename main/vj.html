ENHANCING LIVE PERFORMANCES:
A CO-CREATIVE VJING SYSTEM USING GENERATIVE AI AND REAL-TIME INPUTS

In collaboration with
Sanne Dekker, Huien Tan
& Carita Muhli
Three still frames from the video output, prompt “Minecraft”, step ~0.6.

Project overview

This project was developed as part of the Computational Creativity course in the Media Technology master’s program at Leiden University. It explores the creative potential of generative AI in live performances by developing a co-creative VJing system that generates visuals in response to real-time camera input and text prompts.

Concept and Goals

The system aims to support VJs and DJs by integrating generative AI into live visual performance workflows. It allows users to collaborate with AI to create visuals that evolve in real time. The goal was to investigate how such a system could enhance creativity, and how generative visuals produced through human-AI collaboration might introduce novel workflows and outputs within the context of VJing.

System Design

The system combines StreamDiffusion (a real-time diffusion model), camera input, and TouchDesigner. It processes live video, denoises it, and generates visual output based on text prompts and camera imagery. Users can adjust the similarity between the prompt and output using a “step” slider. A web interface was developed to make the system accessible to users unfamiliar with TouchDesigner.

Evaluation

The system was evaluated through in-person user tests and a questionnaire, focusing on Ritchie’s empirical criteria: novelty, typicality, and quality. Five participants with backgrounds in VJing, DJing, or visual art interacted with the system, entered prompts, adjusted parameters, and reflected on their experience.

Outcome and Reflections

Participants described the system as creatively inspiring, with visuals that were often considered novel and not achievable with existing tools. While some saw the AI as a creative partner, others viewed it as a tool. Limitations included frame rate constraints, prompt sensitivity, and restricted real-time manipulation. Nonetheless, the system showed potential for expanding visual creativity in live contexts and prompted reflections on the collaborative role of AI in artistic practice.

Just for Fun

After the course ended, I kept using the VJing system—not for research, but just because it was fun. I started playing around with prompts and camera inputs to create strange, sometimes absurd visuals. These experiments had nothing to do with the accuracy of the original project goals. They turned into a kind of visual haiku: deep-fried-like images, meta-memes, alienated in tone, but alive in a line of curiosity.
